## 3. 项目概述

本项目是一个运行在 Windows 11 上的原生 2D 桌宠应用。

v0.0.1 实现透明无边框桌宠、循环待机动画、鼠标拖动、跨显示器移动、显示与隐藏、系统托盘和单实例运行。

## 5. v0.0.1 功能需求

### 5.1 基础显示

| 编号 | 功能 | 实施要求 |
| --- | --- | --- |
| F01 | 透明窗口 | 桌宠以透明窗口显示，只显示宠物图像，不出现矩形底色 |
| F02 | 无边框窗口 | 不显示标题栏、窗口边框和系统窗口按钮 |
| F03 | 始终置顶 | 桌宠保持在普通窗口上方 |
| F04 | 不在任务栏显示 | 桌宠主窗口不占用任务栏按钮 |
| F05 | 隐藏与重新显示 | 可以隐藏桌宠，并通过系统托盘重新显示 |
| F06 | 跨显示器移动 | 用户可以将桌宠拖动到任意已连接显示器 |

### 5.2 动画

| 编号 | 功能 | 实施要求 |
| --- | --- | --- |
| A01 | 待机动画 | 使用一张透明 PNG，通过轻微缩放和上下浮动产生循环待机动画 |

### 5.3 交互

| 编号 | 功能 | 实施要求 |
| --- | --- | --- |
| I01 | 鼠标拖动 | 左键按住桌宠时跟随鼠标移动，松开后停留在当前位置 |

### 5.4 系统功能

| 编号 | 功能 | 实施要求 |
| --- | --- | --- |
| S01 | 系统托盘菜单 | 托盘菜单提供“显示桌宠”“隐藏桌宠”和“退出程序” |
| S05 | 单实例运行 | 同一 Windows 用户会话中只允许存在一个桌宠进程实例 |

### 5.5 功能关系

- F05 通过 S01 实现：桌宠隐藏后，用户通过系统托盘重新显示。
- 隐藏桌宠时执行 P19 所规定的后台节能行为。
- S01 的菜单响应速度满足 P11。

## 8. v0.0.1 架构设计

### 8.1 模块划分

    DesktopPet.App
    ├─ PetWindow
    ├─ IdleAnimationController
    ├─ TrayService
    ├─ SingleInstanceGuard
    ├─ DisplayCoordinateService
    └─ Assets/DefaultPet/pet.png

### 8.2 模块职责

- PetWindow：负责透明无边框窗口、置顶显示和鼠标拖动。
- IdleAnimationController：负责加载宠物图片，并控制轻微缩放和上下浮动的循环待机动画。
- TrayService：负责托盘图标、显示、隐藏和退出菜单。
- SingleInstanceGuard：负责阻止重复启动多个实例。
- DisplayCoordinateService：处理多显示器坐标、负坐标和跨显示器 DPI。
- Assets/DefaultPet/pet.png：保存 v0.0.1 使用的待机动画资源。

### 8.3 实现要求

- 各模块只承担本文档规定的职责。
- 隐藏桌宠时，TrayService 通知 IdleAnimationController 停止动画更新。
- 重新显示桌宠时，恢复待机动画。
- DisplayCoordinateService 为 PetWindow 提供当前显示器和 DPI 信息。
- SingleInstanceGuard 在创建桌宠窗口之前完成实例检查。

## 9. 动画资源要求

v0.0.1 使用用户提供的一张临时桌宠图片，源码路径为：

    src/DesktopPet.App/Assets/DefaultPet/pet.png

资源与实现要求：

- 图片必须使用带透明通道的 PNG 格式。
- 图片文件作为外部内容资源，不把具体图案写死在代码中。
- 程序从固定资源路径加载 pet.png。
- 待机动画由程序对图片进行轻微缩放和上下浮动产生。
- 动画过程中不得出现明显跳动、黑边或矩形背景。
- 更换桌宠图案时，只需要替换 pet.png，不需要修改业务代码。
- 替换后的图片继续使用透明 PNG 格式和相同文件名。

## 11. v0.0.1 验收标准

1. 程序可以在 Windows 11 x64 上启动。
2. 双击程序后 2 秒内显示桌宠，5 秒内完成资源初始化。
3. 桌宠以透明、无边框、置顶窗口显示，并且不在任务栏出现。
4. 程序可以使用一张透明 PNG 持续播放轻微缩放和上下浮动的待机动画。
5. 用户可以使用鼠标拖动桌宠。
6. 桌宠可以跨多个显示器移动。
7. 桌宠能够正确处理负数显示器坐标和 100%、125%、150%、200% DPI。
8. 桌宠跨越不同 DPI 显示器后，大小、位置和图像显示正确。
9. 系统托盘菜单可以显示桌宠、隐藏桌宠和退出程序。
10. 重复启动程序时不会出现第二个桌宠实例。
11. 托盘右键菜单在 200 ms 内出现。
12. 隐藏桌宠后停止动画定时器和透明窗口刷新。
13. 发布版本满足 P01、P02、P04、P05 和 P10 的性能指标。
